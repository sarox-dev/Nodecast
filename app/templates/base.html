Install Recollect Extension

Save highlights from any website to your local Recollect server.

1

Clone the extension repository

Open a terminal and run:

git clone https://github.com/sarox-dev/Recollect-Extension.git

Or download ZIP and extract it.

2

Open Chrome Extensions page

Open chrome://extensions/ in your browser (any Chromium-based browser: Chrome, Edge, Brave, Opera).

💡 Enable Developer mode using the toggle in the top-right corner.
3

Load unpacked extension

Click "Load unpacked" and select the Recollect-Extension folder you just cloned.

4

Pin the extension (optional)

Click the puzzle piece icon 🧩 in your toolbar, find Recollect, and click the pin icon 📌 to keep it visible.

Platform notes

🐧 Linux

Same steps above. Make sure git is installed (sudo apt install git or equivalent).

🪟 Windows

Use Git for Windows or download the ZIP and extract with Explorer.

🍎 macOS

git is pre-installed or get it via xcode-select --install. Otherwise download the ZIP.
